normal_items creates in tail (T)
```
H>|<H:T>|   {1 normal item}
  |  :  |   {2 normal item}
  |  :  |   {3 normal item}
  |  :  |<T {4 normal item}
```
prior_items creates in head (H)
```
H>|  :  |   {6 prior item}
  |<H:  |   {7 prior item}
  |  :T>|   {2 normal item}
  |  :  |   {3 normal item}
  |  :  |<T {4 normal item}
```
